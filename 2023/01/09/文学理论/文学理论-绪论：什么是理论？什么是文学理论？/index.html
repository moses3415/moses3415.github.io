一、何谓理论？

理论约等于哲学
哲学：系统化的世界观和方法论
理论没有那么系统化，理论是对现象的抽象分析和概括

学习理论需要的能力：

  1. 深入思考能力：反思、批判、质疑
    质疑那些不可疑的东西
    Common sense is the collection of prejudices acquired by age eighteen.
    ——Albert Einstein
    常识就是人到十八岁为止所累积的各种偏见。
    E.g. 游戏，如何上瘾？意外的惊喜，朋友圈的比较……
    E.g. app 推送
    E.g. 人的选择是脆弱的，《楚门的世界》

  2. 分析现象背后本质的能力：解读、阐释
    E.g. 同性婚姻合法化，对婚姻的反思，陷入反思的循环，需要解读

乔纳森·卡勒对理论的定义

  1. 理论是分析的话语
  2. 理论是对常识的批评
  3. 理论具有反射性（反思性），是关于思维的思维（冷峻的旁观）
  4. 理论是跨学科的

二、何谓文学理论？

文学的 “哲学”

  1. 作者之意 vs 读者之意
    E.g. 作者做不出根据自己文章出的题目，作者对作品具有绝对权威的阐释权？《安娜·卡列尼娜》

  2. 你是否看到一只鸡？
    克拉考尔《电影的本质》，8 画格的存在，熟悉
    争论结果不重要，重要的是争论的原因，认知的差异

  3. 随意解读 vs 科学的解读
    基于文本

  4. 阅读为何：接受式/反思性
    破坏审美体验，分析的快感
    E.g. 《微微一笑很倾城》，校园爱情的套路，“你出一尺，为挪墙一尺”
    E.g. 武侠小说，建构仇恨
    E.g. 迪士尼动画电影，成长故事
    E.g. 《狮子王》隐性价值观：精英主义价值观，血统论的逻辑

评判作品背后的价值系统

文学理论：谁能写进文学史，地位怎么样
文学理论在不同时代是变动的
当代视角下的王者段位作家：李白、杜甫、陶渊明、苏轼

喜厌和好坏无关系

提供解读作品的角度和方法

一切的意义都是阐释的结果

跨学科性

哲学：史铁生作品的存在主义解读
心理学：精神分析弗洛伊德
语言学、社会学

三、文学理论的研究对象

E.g. 各国地区电视剧鄙视链
